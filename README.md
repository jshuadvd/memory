memory
=========

A small library that provide temporal and durable memory

## Installation

  `npm install --save @usertoken/memory`

## Usage

    var numFormatter = require('@jdaudier/number-formatter');

    var formattedNum = numFormatter(35666);


  Output should be `35,666`


## Tests

  `npm test`
